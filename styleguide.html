Interactive Elements

In-text link

CSS selector:

.un

Sample code:

<a href="#" class="un">text</a>

Rendered element:

text link

Navigation Link

CSS selector:

.nav-links a

Sample code:

<div class="nav-links"><a href="#" class="REM nav-item">Default Button</a></div>

Rendered element:

Home

Submit/Reset Buttons

CSS selector:

.buttonSubmit, .buttonReset

Sample code:

<form> <input class="buttonSubmit" type="Submit"> </form>
<form> <input class="buttonReset" type="Reset"> </form>

Rendered element:

Text Input Field with Label

CSS selector:

.tInput

Sample code:

<form class="tInput"> <input type="text" id="text-input">
<label for="text-input">Label </label></form>

Rendered element:

Radio Button with Label

CSS selector:

.radio

Sample code:

<form class="radio"> <input type="Radio" id="select">
<label for="select">Select</label> </form>

Rendered element:
