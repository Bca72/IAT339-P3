Daisy 7094

Project Overview

Daisy 7094 is an interactive art installation that turns the participants’ conversations into geometric artworks that can then be played back in real time.

Daisy 7094 was created for my Interactive Arts class in the winter of 2022.

Ideation Process

When we were first assigned the task to create an interactive art piece, I wanted participants to be able to walk away from our installation with a customized piece of artwork. After a few more ideation sessions with my group, I came up with the initial idea for Daisy 7094, which was to record a conversation between two random participants and then paint on a canvas using the sound data that we grabbed from microphones.

Design Process

My original idea for the final output was to create an image that was generated based on the voice data of the participants. However, this version of the output ended up looking too generic and the uniqueness of each participant was not being captured. After getting some advice from my professor I realized that the participants should be able to “read” the final output to an extent. If they did not know what the final image represented, then the artwork would lose meaning.
